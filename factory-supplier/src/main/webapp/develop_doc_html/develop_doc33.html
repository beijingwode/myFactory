简易售后订单列表 supplier.refunds.get

a) API描述

名称：简易售后订单列表

描述：获取商家单位售后数据，返回数据结果以售后申请创建时间倒序排列，返回结果仅包含：售后单id、订单id、状态，等信息。更多信息可通过单笔售后部分信息接口获取

数据格式：JSON

请求URL： http://supplier.wd-w.com/open (统一开放接口)

b) 请求参数

名称 类型 必填 说明
appid string 必填 应用唯一标识，我的网商家平台-->首页-->API安全 查看
method string 必填 supplier.refunds.get
timestamp string 必填 时间戳，格式为yyyy-MM-dd HH:mm:ss，时区为GMT+8，例如：2017-01-01 12:00:00
start_created date 非必填 查询三个月内售后申请时间开始。格式:yyyy-MM-dd HH:mm:ss
end_created date 非必填 查询售后申请时间结束。格式:yyyy-MM-dd HH:mm:ss
status string 非必填 售后状态：默认为等待卖家确认（WAIT_SELLER_CONFIRM）
refund_type string 非必填 售后类型: 默认为全部类型
goods_status string 非必填 退货物流状态（退货退款时适用）：默认为全部状态
sign string 必填 API输入参数签名结果，签名算法介绍请点击这里。

注：售后状态（status）可传入以下值：
WAIT_SELLER_CONFIRM：等待卖家确认
REFUND_FINISHED：售后完成
REFUND_CLOSED：售后关闭

注：售后类型（refund_type）说明：
REFUDN_FEE_ONLY：仅退款
RETURN_OF_GOODS：退货并退款

注：退货物流状态（goods_status）说明：
BUYER_SEND_GOODS：买家退货已发出
SELLER_SIGNED_GOODS：卖家已签收退货

c) 返还结果

名称 类型 说明
success bool true：成功 false：失败
message string 失败时，错误信息
data JsonArray 数组元素：
refund_id:售后单id
trade_id:订单id
status:售后状态
refund_type:售后类型
goods_status：退货物流状态 （退货退款时适用）
created:创建时间
payment：订单金额
shipping_fee：订单运费
refund_fee:退款金额
单笔售后部分信息 supplier.refund.get

a) API描述

名称：单笔售后部分信息

描述：获取单笔售后部分信息

数据格式：JSON

请求URL： http://supplier.wd-w.com/open (统一开放接口)

b) 请求参数

名称 类型 必填 说明
appid string 必填 应用唯一标识，我的网商家平台-->首页-->API安全 查看
method string 必填 supplier.refund.get
timestamp string 必填 时间戳，格式为yyyy-MM-dd HH:mm:ss，时区为GMT+8，例如：2017-01-01 12:00:00
refund_id string 选填 售后单id 可通过 简易售后订单列表接口获取 （与订单id 二选一）
trade_id string 选填 订单id 可通过简易订单列表接口获取（与售后单id 二选一）
sign string 必填 API输入参数签名结果，签名算法介绍请点击这里。

c) 返还结果

名称 类型 说明
success bool true：成功 false：失败
message string 失败时，错误信息
data JsonObject json元素：
refund_id:售后单id
trade_id:订单id
status:售后状态
refund_type:售后类型
goods_status：退货物流状态 （退货退款时适用）
created:创建时间
payment：订单金额
shipping_fee：订单运费
refund_fee:退款金额
reason:退款原因
note:退款说明
attachment1:凭证1 图片url
attachment2:凭证2 图片url
attachment3:凭证3 图片url
express_com:快递公司简称 （退货退款时适用）
express_no:快递单号 （退货退款时适用）
skus:{
　　商品信息 （退货退款时适用）
　　sku_id:skuID
　　outer_id:Sku的商家外部id
　　p_title:商品标题
　　spec:规格信息 （例：红色，M）
　　sku_code:条码
　　num:数量
　　price:单价
　　image: 主图url （例：http://img2.wo-de.com/upload//xxx/xx.jpg ）
}
