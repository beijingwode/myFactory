简易订单列表 supplier.trades.get

a) API描述

名称：简易订单列表

描述：获取商家单位交易数据，返回数据结果以订单创建时间倒序排列，返回结果仅包含：订单id、状态，等信息。更多信息可通过单笔交易的部分信息接口获取

数据格式：JSON

请求URL： http://supplier.wd-w.com/open (统一开放接口)

b) 请求参数

名称 类型 必填 说明
appid string 必填 应用唯一标识，我的网商家平台-->首页-->API安全 查看
method string 必填 supplier.trades.get
timestamp string 必填 时间戳，格式为yyyy-MM-dd HH:mm:ss，时区为GMT+8，例如：2017-01-01 12:00:00
start_created date 非必填 查询三个月内交易创建时间开始。格式:yyyy-MM-dd HH:mm:ss
end_created date 非必填 查询交易创建时间结束。格式:yyyy-MM-dd HH:mm:ss
status string 非必填 交易状态：默认为待发货订单（WAIT_SELLER_SEND_GOODS）
sign string 必填 API输入参数签名结果，签名算法介绍请点击这里。

注：交易状态（status）可传入以下值：
WAIT_BUYER_PAY：等待买家付款
WAIT_SELLER_SEND_GOODS：等待卖家发货
WAIT_BUYER_CONFIRM_GOODS：等待买家确认收货
TRADE_BUYER_SIGNED：买家已签收（货到付款专用）
TRADE_FINISHED：交易成功
TRADE_CLOSED：交易关闭

c) 返还结果

名称 类型 说明
success bool true：成功 false：失败
message string 失败时，错误信息
data JsonArray 数组元素：
trade_id：订单id
status：交易状态
created：创建时间
payment：订单金额
单笔交易的部分信息 supplier.trade.get

a) API描述

名称：单笔交易的部分信息

描述：获取单笔交易的部分信息

数据格式：JSON

请求URL： http://supplier.wd-w.com/open (统一开放接口)

b) 请求参数

名称 类型 必填 说明
appid string 必填 应用唯一标识，我的网商家平台-->首页-->API安全 查看
method string 必填 supplier.trade.get
timestamp string 必填 时间戳，格式为yyyy-MM-dd HH:mm:ss，时区为GMT+8，例如：2017-01-01 12:00:00
trade_id string 必填 订单id 可通过 简易订单列表接口获取
sign string 必填 API输入参数签名结果，签名算法介绍请点击这里。

c) 返还结果

名称 类型 说明
success bool true：成功 false：失败
message string 失败时，错误信息
data JsonObject json元素：
trade_id：订单id
status：交易状态
created：创建时间
payment：订单金额
shipping_fee：运费金额
buyer_id：买家id
buyer_name：收货人姓名
buyer_phone：收货人电话
buyer_address：收货地址（包含省、市、区县、详细地址）
buyer_address_province： 收货地址 省
buyer_address_city： 收货地址 市
buyer_address_area： 收货地址 区/县
buyer_address_detail： 收货地址 详细地址
buyer_noto：买家留言
buyer_ invoice：买家发票抬头
express_com：快递公司简称
express_no：快递单号
skus：{ 商品信息
　　sku_id：skuID
　　outer_id：Sku的商家外部id
　　p_title：商品标题
　　spec：规格信息 （例：红色，M）
　　sku_code：条码
　　num：数量
　　price：单价
　　image： 主图url （例：http://img2.wo-de.com/upload//xxx/xx.jpg ）
}

名称：订单发货处理

描述：将快递公司及快递单号更新至订单，并将订单状态自动修改“卖家已发货”

数据格式：JSON

请求URL： http://supplier.wd-w.com/open (统一开放接口)

b) 请求参数

名称 类型 必填 说明
appid string 必填 应用唯一标识，我的网商家平台-->首页-->API安全 查看
method string 必填 supplier.trade.express.send
timestamp string 必填 时间戳，格式为yyyy-MM-dd HH:mm:ss，时区为GMT+8，例如：2017-01-01 12:00:00
trade_id string 必填 订单id 可通过简易订单列表接口获取
express_com string 必填 快递公司：简拼代码或简称 可通过快递公司列表接口获取
express_no string 必填 运单号.具体一个物流公司的真实运单号码。系统会根据运单号自动追踪物流状态
sign string 必填 API输入参数签名结果，签名算法介绍请点击这里。

c) 返还结果

名称 类型 说明
success bool true：成功 false：失败
message string 失败时，错误信息
data JsonArray 未定
