简易商品列表 supplier.products.get

a) API描述

名称：简易商品列表

描述：获取商家单位，在售商品列表简易信息，包括商品id：p_id、商品标题：p_title、商品分类：p_category

数据格式：JSON

请求URL： http://supplier.wd-w.com/open (统一开放接口)

b) 请求参数

名称 类型 必填 说明
appid string 必填 应用唯一标识，我的网商家平台-->首页-->API安全 查看
method string 必填 supplier.products.get
timestamp string 必填 时间戳，格式为yyyy-MM-dd HH:mm:ss，时区为GMT+8，例如：2017-01-01 12:00:00
sign string 必填 API输入参数签名结果，签名算法介绍请点击这里。

c) 返还结果

名称 类型 说明
success bool true：成功 false：失败
message string 失败时，错误信息
data JsonArray 数组元素：
p_id：商品id
p_title：商品标题
p_category：商品分类
outer_id：商品外部编码
SKU列表(无图片) supplier.product.skus.get

a) API描述

名称：商品SKU列表

描述：获取商品单位，SKU列表简易信息，包括sku id:sku_id、价格：price、库存：quantity、规格：spec 多个规格值时以逗号分隔

数据格式：JSON

请求URL： http://supplier.wd-w.com/open (统一开放接口)

b) 请求参数

名称 类型 必填 说明
appid string 必填 应用唯一标识，我的网商家平台-->首页-->API安全 查看
method string 必填 supplier.product.skus.get
p_id string 选填 商品id 可通过 简易商品列表接口获取 （与商品外部编码，p_title三选一）
outer_id string 选填 商品外部编码（与p_id，p_title三选一）
p_title string 选填 商品标题 可通过 简易商品列表接口获取（与outer_id ,p_id三选一）
timestamp string 必填 时间戳，格式为yyyy-MM-dd HH:mm:ss，时区为GMT+8，例如：2017-01-01 12:00:00
sign string 必填 API输入参数签名结果，签名算法介绍请点击这里。

注：可以通过商品id 或者商品标题获取sku 信息。但商品标题必须能过唯一确定一条商品

c) 返还结果

名称 类型 说明
success bool true：成功 false：失败
message string 失败时，错误信息
data JsonArray 数组元素：
sku_id:sku id
outer_id:Sku的商家外部id
price:sku 售价
quantity:库存
spec:规格值 多个规格值时以逗号分隔 例：红色，M
image:主图url （例：http://img2.wo-de.com/upload/regular/xxx/xx.jpg ）
SKU库存更新 supplier.product.sku.upd

a) API描述

名称：SKU库存更新

描述：SKU库存更新

数据格式：JSON

请求URL： http://supplier.wd-w.com/open (统一开放接口)

b) 请求参数

名称 类型 必填 说明
appid string 必填 应用唯一标识，我的网商家平台-->首页-->API安全 查看
method string 必填 supplier.product.sku.upd
sku_id string 选填 Sku id 可通过 SKU列表接口获取 （与sku的商家外部id ，sku信息三选一）
outer_id string 选填 Sku的商家外部id（sku_id ，sku信息 三选一）
p_title string 选填 商品标题 可通过 简易商品列表接口获取（与spec组成sku信息，并与sku_id，Sku的商家外部id三选一）
spec string 选填 sku规格 可通过 SKU列表接口获取 （与p_title组成sku信息，并与sku_id，Sku的商家外部id三选一）
quantity long 必填 最新库存量，直接更新该sku 库存
timestamp string 必填 时间戳，格式为yyyy-MM-dd HH:mm:ss，时区为GMT+8，例如：2017-01-01 12:00:00
sign string 必填 API输入参数签名结果，签名算法介绍请点击这里。

注：可以通过商品标题+sku规格更新sku信息。但商品标题+sku必须能过唯一确定一条sku

c) 返还结果

名称 类型 说明
success bool true：成功 false：失败
message string 失败时，错误信息
data long 返回更新后数量

outer_id string 必填 商品外部编码,长度不能超过64字节
timestamp string 必填 时间戳，格式为yyyy-MM-dd HH:mm:ss，时区为GMT+8，例如：2017-01-01 12:00:00
sign string 必填 API输入参数签名结果，签名算法介绍请点击这里。

注：可以通过商品标题获进行设置。但商品标题必须能过唯一确定一条商品

c) 返还结果

名称 类型 说明
success bool true：成功 false：失败
