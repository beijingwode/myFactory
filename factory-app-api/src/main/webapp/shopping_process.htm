购物流程

注册

买家app没有注册入口，需员工邀请。

买家app没有注册入口，所有账号需要通过企业添加员工或者员工邀请好友。新注册的账号初始密码跟帐号相同，首次登录需重置密码。员工邀请好友的3个入口:

1、朋友圈→亲友管理→添加亲友，发送亲友邀请后对 方会收到短信提醒或系统提示（已有账号）

2、商品详情/购物车→分享→亲友邀请分享（购物车无此步骤）→分享到微信/微信扫描生成的二维码→微信长按识别分享页二维码→绑定亲友

3、招商员工（限我的福利员工）登录，个人中心→招商二维码→他人扫码成功后，在绑定亲友页面可以选择绑定手机亲友或临时账号亲友，立即绑定后可以浏览网页版app系统，如要购买商品需安装下载

查找商品

分类浏览或者直接搜索来查找商品

商品查找方式有两种：

1、搜索商品

买家登录，系统首页或分类页搜索框中输入搜索词，搜索相关商品。

2、分类浏览

选择要查看的分类名称，点击显示该分类的所有商品，搜素结果页面点击筛v选可以根据条件缩小查询范围。

放入购物车

将商品放入购物车等待购买。

商品加入购物车有两种方式：

1、商品详情页加入购物车，如需修改商品规格可以在规格选择页内加入购物车。

2、已关闭和已收货的订单，列表或详情页面中可以“再次购买”订单中的所有商品。

提交订单

选择配送地址及优惠券使用方式后提交订单。

买家可由商品详情页面直接购买或购物车结算两种方式下单，订单确认页面选择配送地址及优惠券的抵扣方式，若用券后订单合计总金额为0，提交订单有确认支付询问，确定后订单支付成功；若用券后订单合计总金额大于0，提交订单后选择在线支付方式继续支付。

查看订单状态

买家登录个人中可查看订单状态及物流信息。

收货后评价

买家收到货后修改订单状态并对商品做评价。

待收货状态的订单，列表或订单详情页面都有确认收货按钮，买家收到货后可将订单状态修改成已收货。已收货状态的订单买家可以进行商品评价。

售后维权

买家收货后，如有需要可以申请售后。

买家对购买商品有质疑，可以确认收货后再申请售后，提交申请后等待商家审核。
