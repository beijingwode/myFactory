怎样才能使用代收货服务？
  • 您在下单填写地址时可勾选离您最近的我的代收点为您提供该服务，您可以通过下单页面的地图或者“我的快递”APP查询您想要使用的代收货服务站点。非线上选择（如：手动复制驿站地址为默认地址）等情况，不支持使用免费代收货服务。

    因站点场地限制，如代收的包裹体积过大，单个包裹最长边超过55cm或单个包裹合计尺寸（长+宽+高）超过110cm、 圆卷型包裹长度超过90cm或直径和长度合计超过110cm，站点将可视自身实际情况签收或者拒签（请先行和商家沟通，确定包裹尺寸，避免因包裹过大，导致代收货出现异常）
