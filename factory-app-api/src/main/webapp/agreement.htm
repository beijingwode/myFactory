2.5 “消费者”：指符合条件并在“我的网平台”上购买商品的企业用户（商家）及个人用户（员工）。

2.8 “店铺”：指企业用户在完成用户注册流程后，为进行合法经营，依据相关协议约定和“我的网平台”规则由企业用户申请，我的网审核通过的具有独立且唯一性ID（英文Identity的缩写，意为“身份标识号码”,此处指“用户ID”）、特定名称（可依据相关平台规则进行调整）的网络化虚拟商铺。

2.9 “商品”：为表述便利，在本协议中，“商品”可能指代商品和服务的合称。

3.3.1 在用户下订单时，请仔细确认销售方（即商家）、所购商品的名称、价格、数量、型号、规格、尺寸、联系地址、电话、收货人等信息。收货人与用户本人不一致的，收货人的行为和意思表示视为用户的行为和意思表示，用户应对收货人的行为及意思表示的法律后果承担连带责任。

3.3.2 除法律另有强制性规定外，双方约定如下：本站上销售方展示的商品和价格等信息仅仅是交易信息的发布，用户下单时须填写希望购买的商品数量、价款及支付方式、收货人、联系方式、收货地址等内容；系统生成的订单信息是计算机信息系统根据用户填写的内容自动生成的数据，仅是用户向销售方发出的交易诉求；销售方收到用户的订单信息后，只有在销售方将用户在订单中订购的商品从仓库实际直接向用户发出时（以商品出库为标志），方视为用户与销售方之间就实际直接向用户发出的商品建立了交易关系；如果用户在一份订单里订购了多种商品并且销售方只给用户发出了部分商品时，用户与销售方之间仅就实际直接向用户发出的商品建立了交易关系；只有在销售方实际直接向用户发出了订单中订购的其他商品时，用户和销售方之间就订单中该其他已实际直接向用户发出的商品才成立交易关系。用户可以随时登录用户在本站注册的账户，查询用户的订单状态。

3.4.1 销售方将会把商品送到用户所指定的收货地址，所有在本站上列出的送货时间为参考时间，参考时间的计算是根据库存状况、正常的处理过程和送货时间、送货地点的基础上估计得出的。

（4）不可抗力因素导致的，例如：自然灾害、交通戒严（或管制）、突发战争等。
