支付流程

买家提交订单分两种情况：提交订单时不需再支付（提交时订单合计总金额为0）和提交订单时需支付（提交订单时订单合计总金额大于0）。

1、提交订单时不需再支付的订单，买家提交订单时会弹出支付确认对话框，买家选择确认支付成功，选择取消返回确认订单页面。

2、提交订单时需支付的订单，买家提交订单时跳到在线支付页面，选择余额支付（现金余额够支付订单），确认支付后订单支付成功。如果买家账户余额不足，该支付方式不可选。

在线支付页面选择支付宝或微信方式支付，如已安装客户端直接跳到对应的客户端支付，未安装客户端，选择支付宝支付可以跳到网页版继续操作，选择微信支付提示用户未安装。

说明：

订单不需再支付：是买家在确认订单页面使用资产中的内购券、现金券和换领币（购买换领商品，且买家自己有换领币）抵扣商品售价，最终使订单的合计金额为0。

换领币使用规则：

如果买家账户有换领币，购买换领商品可以使用换领币抵扣。

单个商品换购券使用数量上限等于商品内购价，内购券、邮费和内购券不足部分都需要现金补齐；如果换领币不足只能使用现金补齐。
